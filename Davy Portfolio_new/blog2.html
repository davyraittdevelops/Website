Experimenting with Runecrafting Automation in Old School RuneScape

A Nostalgic Experiment

I used to spend countless hours on Old School RuneScape, immersed in its vast world. Although I don't play much these days, programming has become one of my new hobbies. So, driven by nostalgia and a newfound passion for coding, I decided to experiment with creating an automated runecrafting script using Tribot. This project was purely for fun and to challenge myself—it was about merging my old love for the game with my new interest in automation.

The Script: V1_DavyF2PRunecrafter

The outcome of this nostalgic fusion is "V1_DavyF2PRunecrafter," a script that automates the crafting of all F2P runes in Old School RuneScape. It even handles the logistics of buying supplies, making it a comprehensive automation tool.

Challenges and Learning Points

Creating this script was not a walk in the park. It involved deep diving into game mechanics, handling unexpected scenarios, and integrating complex APIs. This journey taught me resilience, the importance of debugging, and gave me a firsthand experience in creating non-trivial automation scripts.

Disclaimer and Ethical Considerations

It’s important to note that botting is against the rules in Old School RuneScape and can result in permanent bans. Therefore, I will not be using this script on any live accounts. This project was conducted purely for educational purposes to satisfy my curiosity and passion for programming.

Closing Thoughts

This project was an exciting way to reconnect with a game I loved while indulging in my passion for programming. For those interested in game scripting or automation, I recommend pursuing similar projects for learning and fun—just remember to respect game rules and use scripts ethically.
