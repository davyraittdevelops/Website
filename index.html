Mobile guiding system.

In period 2.2 we created an Android App which you can use to walk routes trough Breda. A route contains multiple points of interest. You can select these points and check out a demo video, or read some general information. It also contains a picture of the POI. In the app you can also change the language, text size, both of these instantly refresh. You can also enable color blind modus. Ultrasonic Atmega 128 (C)

In period 2.3 we had the course "Microcontrollers". In this course we worked with the Atmel AVR Atmega 128. Which is a CMOS 8 bit microcontroller. I attached an ultrasonic sensor (HC-SR04) to the Microcontroller and with it I measured the distance and displayed it to an LCD. The drivers for the Ultrasonic sensor and LCD were both written from scratch. Notable parts of the application are: Pre processor code, hardware interrupts, timers, LCD controller, controlling US with pulses, and calculating distances.

You can check out a demo video with this link.

Vision/Machine learning(C++)

For a side project in period 2.3 we had to use OpenCV. We choose a final assignment in which we had to automatically recognize russian license plates. By using the correct pre-processing in OpenCV we could clean up the images and morph them into very easily readable license plates. All we had to do after that is find the contours and save them to seperate files. I also used Tesseract OCR to recognize the characters on the image. I could feed TOCR an existing dataset and by using a neural network it would use this to read the license plate.

You can check out a demo video with this link.
